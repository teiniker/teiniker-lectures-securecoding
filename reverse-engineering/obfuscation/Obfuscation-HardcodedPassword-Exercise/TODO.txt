Exercise: Obfuscation - Hardcoded Password
-------------------------------------------------------------------------------

Given an obfuscated JAR file (HardcodedPassword-Dev.jar), we can run the example
like:
$ java -cp HardcodedPassword-Release.jar org.se.lab.Main
password> xxxx
Login rejected, invalid password!

A) Decompile the generated Release File
- Use https://www.decompiler.com/ for JDK21 code

B) Extract hardcoded password and try to login:
$ java -cp HardcodedPassword-Dev.jar org.se.lab.Main
password>
